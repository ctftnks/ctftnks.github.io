# 12. Glossary

| Term       | Definition                                                      |
| :--------- | :-------------------------------------------------------------- |
| **Bot**    | An AI-controlled player.                                        |
| **Canvas** | HTML5 element used for rendering 2D graphics.                   |
| **CTF**    | Capture The Flag (Game Mode).                                   |
| **Entity** | A dynamic object in the game world (Tank, Bullet).              |
| **Frame**  | A single iteration of the game loop and render cycle.           |
| **Tick**   | A single logic update step.                                     |
| **Tile**   | A static square unit of the map grid. Can be a floor or a wall. |
| **Vue**    | The JavaScript framework used for the UI.                       |
| **Store**  | Refers to Pinia/Vue stores used for state management.           |
