# 2. Architecture Constraints

## 2.1 Technical Constraints

- **Platform:** Modern Web Browsers (Chrome, Firefox, Safari, Edge).
- **Language:** TypeScript for logic, Vue 3 for UI components.
- **Distribution:** Static site hosting (specifically GitHub Pages).
- **No Backend:** The game logic runs entirely client-side. Multiplayer is currently local (bots) or potentially sharing the same screen/device (though controls suggest single keyboard usage primarily for now).

## 2.2 Organizational Constraints

- **Open Source:** The project is hosted on GitHub.
- **Team Size:** Small team/individual project.
- **License:** ISC License.

## 2.3 Conventions

- **Code Style:** Prettier and ESLint are used for formatting and linting.
- **Testing:** Vitest is used for unit testing.
